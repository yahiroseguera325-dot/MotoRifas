Moto Rifas Universal - Node.js demo

Requisitos:
- Node.js instalado (v14+ recomendado)

Instalación y correr localmente:
1. Descomprime el proyecto.
2. Abre una terminal en la carpeta del proyecto (donde está server.js y package.json).
3. Ejecuta: npm install
4. Ejecuta: npm start
5. Abre en el navegador: http://localhost:3000/comprar.html
